


### 1. Clone & cài packages
```bash
git clone <repo-url>
cd orm-dashboard
npm install
```
### 2. Tạo file .env
Điền vào `.env:
```
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...
GEMINI_API_KEY=
GOOGLE_PLACES_API_KEY=AIza... # tuỳ chọn
```
### 3. Chạy local
```bash
npm run dev
# Mở http://localhost:3000
```
